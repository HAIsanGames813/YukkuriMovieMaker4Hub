# YukkuriMovieMaker4Hub
ゆっくりムービーメーカー4のハブソフトです

### 導入方法
#### [最新バージョンはこちらから](https://github.com/HAIsanGames813/YukkuriMovieMaker4Hub/releases/latest)
1.`YMM4Hub.msi`をクリックしてダウンロード  
2.ダウンロード後msiファイルを開きインストール  
**注意**:プログラムフォルダに置くと管理者として実行する必要があるためプログラムフォルダ以外(Userなど)に配置するようにしてください

## 概要
YMM4を前回のプロジェクト、新規プロジェクト、ymmpから開くことができる外部ソフトです  
## 使い方
### インスタンスの追加方法
※3.0.0以降で有効です  
### 3.0.0、3.1.0での手順  
インスタンス欄の下部にある追加から既存のYukkuriMovieMaker.exeを選択し追加します  
### 4.0.0以降での手順  
1、インスタンス欄の下部にある追加から新規ダウンロード、または既存を追加を選択
#### 新規ダウンロードを選択した場合
2、通常版かLite版かを選択  
3、インスタンス名、インスタンスアイコンを設定  
4、インストール先のパスを指定  
5、ダウンロード&&作成を選択  
#### 既存を追加を選択した場合
2、既存を追加を選択した場合はYukkuriMovieMaker.exeを選択  
3、インスタンス名、インスタンスアイコンを設定  
4、追加を選択  

### プロジェクトファイルリストの設定
※3.0.0以降で有効です  
1、Hub設定タブを開く  
2、編集データフォルダの監視で.ymmp、.ymmpx、.ymmeが入っているフォルダを選択(複数設定可能)  

### プラグイン管理やプラグインポータルでより多くの情報を表示する
※4.0.0以降で有効です  
既存のプラグインを一度ダウンロードし直すことでプラグインフォルダ内にinfo.jsonを生成できます  
注意:Githubで公開されていないプラグインは情報の取得ができないため表示できません
