# YukkuriMovieMaker4Hub
ゆっくりムービーメーカー4のハブソフトです

## 導入方法
ダウンロードしたファイルをダブルクリックして、表示される画面に従ってインストールしてください。
### [最新バージョンはこちらから](https://github.com/HAIsanGames813/YukkuriMovieMaker4Hub/releases/latest)

## 概要
YMM4を前回のプロジェクト、新規プロジェクト、ymmpから開くことができる外部ソフトです

## 使用方法
解凍したファイルの中の`YMMHub.exe`を起動します

# 機能説明
## メイン画面
### 前回のプロジェクトを開く
直近に保存されたプロジェクトを開きます

### 新規プロジェクトの作成
新規プロジェクトを作成の状態で開きます

### ymmpから開く
開かれたウィンドウからymmpを選択しプロジェクトを開きます  
ymmpから開く場合は関連付けがされているYMMで起動します

## 設定画面
### 外観
ライト、ダーク、ブラックから選択できます
※ダーク、ブラックは一部変更されません

### フォント
Hubで使用するフォントを変更できます

### YMM4実行ファイルパスを指定
起動するYMM4のパスを指定できます

### 自動終了
・前回のプロジェクトを開く  
・新規プロジェクトの作成  
・ymmpから開く
を選択したときにHubを自動で閉じるかの選択ができます
