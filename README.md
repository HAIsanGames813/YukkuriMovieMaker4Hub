# YukkuriMovieMaker4Hub
ゆっくりムービーメーカー4のハブソフトです

## 導入方法
ダウンロードしたファイルを解凍し、フォルダ内のexeファイルを起動してください
### [最新バージョンはこちらから](https://github.com/HAIsanGames813/YukkuriMovieMaker4Hub/releases/latest)

## 概要
YMM4を前回のプロジェクト、新規プロジェクト、ymmpから開くことができる外部ソフトです

## 使用方法
解凍したファイルの中の`YMMHub.exe`を起動します

# 機能説明
## 起動
### 起動
パス指定されたymmを開きます

### 前回のプロジェクト
パス指定されたymmで前回開かれたプロジェクトを開きます

### 新規作成
パス指定されたymmで新規プロジェクトを作成の状態で開きます

### フォルダマーク
パス指定されたymmで選択したプロジェクトファイルを開きます  

### 歯車マーク
設定画面を開きます

## プラグインポータル
### 検索機能
プラグイン名や作者で検索できます

### 選択/選択解除
ダウンロードするプラグインを選択できます

### 選択したプラグインを一括ダウンロード
選択したプラグインが配布されているサイトを一括で開きます  
※今後パス指定されたymmに自動ダウンロードする機能を搭載します

### プラグイン詳細
左側のリストで名前をクリックしたプラグイン詳細を右側の余白に表示します  
プラグイン名、作者名、公開日、プラグインの説明が書かれています  
(情報はすべてhttps://manjubox.net/ymm4plugins.yml 及び  
https://manjubox.net/api/ymm4plugins/github/list から取得しています)

## 設定
### YMM4実行ファイルパス
起動するYMM4のパスを指定できます  

### テーマ設定
ライト、ダーク、ブラックから選択できます
※ダーク、ブラックは一部変更されません

### プラグイン並び順
公開日順、名前順から選べます
チェックをつけることで降順(Z-A、新しい順)、外すことで昇順(A-Z、古い順)にできます

### フォント設定
Hubで使用するフォントを変更できます

### 自動的に閉じる
・起動  
・前回のプロジェクト  
・新規作成  
・フォルダマーク(ymmp選択)
を選択したときにHubを自動で閉じるかの選択ができます
