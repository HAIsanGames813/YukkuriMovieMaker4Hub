# YukkuriMovieMaker4Hub
ゆっくりムービーメーカー4のハブソフトです

## 導入方法
### [最新バージョンはこちらから](https://github.com/HAIsanGames813/YukkuriMovieMaker4Hub/releases/latest)
1.`YukkuriMovieMaker4Hub.zip`をクリックしてダウンロード  
2.ダウンロード後zipファイルを解凍する  
3.解凍してできたフォルダをPC上の自由な場所に移動  
4.フォルダ内の`YukkuriMovieMaker4Hub.exe`を起動  

## 概要
YMM4を前回のプロジェクト、新規プロジェクト、ymmpから開くことができる外部ソフトです

# 機能説明
## インスタンス
|マーク/言葉|説明|
|-|-|
|▶|YMM4を起動します|
|🕓️|前回のプロジェクトを開きます|
|＋|新規プロジェクトを起動後すぐに作成します|
|＋ 追加|インスタンスを追加します|
|🗑️ 削除|選択中のインスタンスを削除します|

## 概要
|マーク/言葉|説明|
|-|-|
|(四角の空白部分)|選択中のYMM4のインスタンスアイコンを設定できます|
|インスタンス名|選択中のYMM4のインスタンス名を変更できます|
|▶ 起動|選択中のYMM4を起動します|
|📁 フォルダ|選択中のYMM4のフォルダを開くことができます|
|🕓️ 前回のプロジェクトを開く|選択中のYMM4で前回のプロジェクトを開く事ができます|
|＋ 新規作成|選択中のYMM4で新規プロジェクトを作成することができます|
|プロジェクト･ファイル名で検索... 🔍️|ymmp/ymmpx/ymmxファイルを検索できます|
|☑ .ymmp|ymmpのみ表示にします|
|☑ .ymmpx|ymmpxのみ表示にします|
|☑ .ymmx|ymmxのみ表示にします|
|フィルタ解除|検索をリセットしすべての☑をONにします|
|以下はドラッグすることで移動が可能です|
|ファイル名|下にファイル名が並びます。プロジェクト名をクリックすることで選択中のYMM4で開くことができます|
|最終更新日時|下にファイルの最終更新日時が並びます|
|サイズ|下にファイルの容量が並びます|
|拡張子|下にファイルの拡張子が並びます|

## プラグイン管理
|マーク/言葉|説明|
|-|-|
|検索:|プラグイン名で検索ができます|
|選択項目を有効/無効|選択中のプラグインの有効無効の切り替えができます|
|選択項目を削除|選択中のプラグインの削除ができます|
|プラグイン名|下に選択中のYMM4に入っているプラグインの名前が並びます|
|状態|下にプラグインの有効か無効かの表示が並びます|
|アクション/切替|個別に有効無効の切り替えができます|
|アクション/削除|個別に削除ができます|

## プラグインポータル
|マーク/言葉|説明|
|-|-|
|プラグイン検索:|プラグイン名で検索ができます|

## Hub設定
|マーク/言葉|説明|
|-|-|
|☑YMM4起動時にHubを終了する|YMM起動時に自動的にHubを閉じます|
|テーマ設定/Windows|HubのテーマをOSのテーマ設定に自動的に変わります|
|テーマ設定/Light,Dark,Black|Hubのテーマをライト,ダーク,ブラックに設定できます|
|フォントを検索...|フォントを検索できます|
|日本語フォントのみ|フォントを日本語のもののみにフィルタリングします|
|(フォント名)|デバイス内のフォントから選択しHubのフォントを変更することができます|
|フォルダを追加|概要で表示するプロジェクトデータの保存先を追加することができます|
|削除|プロジェクトデータの表示元から削除できます|
|トークン:|プラグインポータルでのデータ取得時の制限を緩和することができます。未入力でも使用できますがすぐに取得できなくなります|
|トークンを発行する|トークンを発行するためのサイト(github)へアクセスします。トークンの発行にはgithubアカウントが必要です|
